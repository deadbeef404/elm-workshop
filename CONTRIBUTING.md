## Authoring

Install [Hugo](https://gohugo.io)

Install the theme:

```
git clone git@github.com:vjeantet/hugo-theme-docdock.git themes/docdock
```

### Preview

```
hugo serve
```

### Build

```
hugo
```

### Editing

All content for the worshop is in `/content`
